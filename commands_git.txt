Чтобы инициализировать git в свой проект
git init



Чтобы посмотреть на версию гит
git --version

Перемещение по проекту:
чтобы зайти в какую-то директорию - cd test

Чтобы выйти из директории - cd ..

Чтобы показать содержимое нашей директории
ls

Чтобы в терминале открыть ФАЙЛ
nano commands_git.txt

Статус нашего git
git status

Чтобы добавить мои вещи в GIT-Репозитрий
git add название_файла_который_добавляем_в_гит - Добавляет в гит указанный файл
git add . - Добавляем все файлы в гит


Чтобы сохранить сделать снимок нашего проекта мы будем использовать команду
Т.е сохранить наш проект в GIT
git commit -m 'Описание того, что у нас случилось'
